О нас

Мы — команда профессионалов, стремящихся предоставить лучшие решения для разметки изображений. Наш инструмент предназначен для создания датасетов и аннотирования изображений.

Информация

Данный проект разрабатывается как web-сервис для разметки изображений и дальнейшего экспорта. Сервис позволяет загружать папку с изображениями, выделять области с помощью различных инструментов и сохранять аннотации.

📚 Краткое руководство:

  • Загрузка: выберите "Папка с изображениями" → укажите папку
  • Инструменты: прямоугольник, круг, произвольная область
  • Рисование: зажмите левую кнопку мыши и тяните
  • 💾 Сохранить разметку: сохраняет JSON и фрагменты
  • Очистить: удаляет текущую разметку
  • Следующее: переход к следующему изображению
  • 📦 Экспорт всех: сохраняет все разметки сразу
  • ➕/➖: масштабирование изображения
  • 🧪 Тест координат: проверка системы координат
